# Button

The Button component is used to trigger an action or event, such as submitting a form,
opening a Dialog, canceling an action, or performing a delete operation.

## React Component

#### Import library

```
Import {Atoms} from '@bonjour-les-pixels/component-library';
```

#### Use Buttom component

```
<Atoms.Button {...props}> CHILDREN_TEXT </Atoms.Button>
```

### ClassName

You can use the `className` prop to override a component's styles using custom classes.

```
  <Button
  className="override-style-componenent"
/>
```

### HandleClick

You can use the `HandleClick` prop for determine what action is to be taken whenever an event is trigger.
This prop is required.

```
  <Button
  HandleClick={() => alert('event trigger')}
/>
```

### Size

You can use the size prop to modify the button size. You can set the value to `xsmal`, `small`, `medium` or `default`. 
The default size is default and therefore doesn't need to be specified.

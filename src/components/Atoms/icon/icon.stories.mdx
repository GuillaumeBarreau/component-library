# Badge

The Icon are visual symbols for communicating additional meaning, status, and available actions.

## React Component

#### Import library

```
Import {Atoms} from '@bonjour-les-pixels/component-library';
```

#### Use Icon component

```
<Atoms.Icon {...props}/>
```

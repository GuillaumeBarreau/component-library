# Link

Use a Link when you want the user to navigate to another page, a specific location on a page, or to download a document.

## React Component

#### Import library

```javascript
Import {Atoms} from '@bonjour-les-pixels/component-library';
```

#### Use Link component

```javascript
<Atoms.Link {...props}> CHILDREN_TEXT </Atoms.Link>
```

### Notice

You can use the `notice` prop to modify the style color. This props can be declined in 5 ways. The style varies according to the level of importance of the action or the brand color in the page context.

### ClassName

You can use the `className` prop to override a component's styles using custom classes.

```
  <Alert className="OVERRIDE_STYLE_CLASSNAME" />
```

### Size

You can use the size prop to modify the button size. You can set the value to `xsmal`, `small`, `default` or `medium`. 
The default size is default and therefore doesn't need to be specified.

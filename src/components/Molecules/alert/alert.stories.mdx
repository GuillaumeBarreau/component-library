# Alert

The Alert component is a small text box notifying the user with short, timely, and relevant information. 
The Alert component appears at the bottom-right of the page.

## React Component

#### Import library

```
Import {Molecules} from '@bonjour-les-pixels/component-library';
```

#### Use Buttom component

```
<Molecules.Alert {...props}> CHILDREN_TEXT </Molecules.Alert>
```

### Notice

You can use the `notice` prop on a Alert to modify the background color. You can set the value to primary, success, warning, info or error. The style varies according to the level of importance of the action in the system context.

### ClassName

You can use the `className` prop to override a component's styles using custom classes.

```
  <Alert className="OVERRIDE_STYLE_CLASSNAME" />
```
